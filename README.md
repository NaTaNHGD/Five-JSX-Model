# ⚛️ FiveM React JSX Boilerplate

Boilerplate para desenvolvimento de interfaces NUI no FiveM utilizando React (JSX).

---

## 📦 Descrição

Este projeto fornece uma base moderna e otimizada para criação de interfaces NUI (Native UI) no FiveM usando React. Ele facilita o desenvolvimento de UIs interativas, organizadas e escaláveis, substituindo HTML/CSS/JS puro por uma abordagem mais robusta.

---

## 🚀 Features

- ⚛️ React com JSX
- 🔥 Hot Reload
- 📦 Build otimizado para produção
- 🔌 Comunicação simples com NUI
- 🎨 Estrutura escalável
- 💡 Compatível com Tailwind, CSS Modules, etc.
