# Five-JSX-Model
Este boilerplate fornece uma base moderna e otimizada para desenvolvimento de interfaces NUI (Native UI) no FiveM utilizando React (JSX). Ele foi criado para facilitar a criação de interfaces interativas, organizadas e escaláveis dentro de recursos do FiveM, com suporte a hot reload, build otimizado e comunicação simplificada.

# ⚛️ FiveM React JSX Boilerplate

Boilerplate para desenvolvimento de interfaces NUI no FiveM utilizando React (JSX).

## 🚀 Features

- ⚛️ React com JSX
- 🔥 Hot Reload para desenvolvimento rápido
- 📦 Build otimizado para produção
- 🔌 Integração simples com NUI (SendNUIMessage / fetch)
- 🎨 Estrutura organizada e escalável
- 💡 Pronto para Tailwind, CSS Modules ou Styled Components (opcional)
